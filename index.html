☁️ iCloud 동기화 설정

Apple CloudKit을 통해 아이폰 ↔ 맥북 실시간 동기화.
Apple Developer 계정 무료 플랜으로 가능.

설정 방법 (한 번만):
1. developer.apple.com → 무료 가입
2. icloud.developer.apple.com 접속
3. + 컨테이너 추가 → 이름: iCloud.daeri.ledger
4. 컨테이너 클릭 → API 액세스 → 토큰 생성 (웹 직접 접근)
5. 아래에 붙여넣기

☁️ Google Drive Auto-Backup

매일 자동으로 Google Drive에 백업해요.
같은 파일을 덮어쓰기하므로 파일이 쌓이지 않아요.
Drive 버전 히스토리로 이전 날 데이터도 복원 가능해요.

설정 방법:
1. Google Cloud Console → 새 프로젝트
2. Google Drive API 사용 설정
3. OAuth 2.0 클라이언트 ID 생성 (Web)
4. 승인된 JS 출처에 파일 경로 추가
5. 아래에 Client ID 붙여넣기
